# 括弧列区間クエリ

## 問題文

長さ $N$ の文字列 $S$ が与えられます。  
$S$ の各文字は `'a'`〜`'z'`、`'('`、`')'` のいずれかです。

$Q$ 個のクエリを順に処理してください。  
クエリは次の 2 種類です。

- `1 i c`  
  $S$ の $i$ 文字目を文字 $c$ に変更します。

- `2 l r`  
  部分文字列 $T = S[l..r]$ を取り出します。  
  $T$ から `'('` と `')'` 以外の文字をすべて削除して得られる文字列を $U$ とします。  
  $U$ が正しい括弧列なら `Yes`、そうでなければ `No` を出力してください。

ここで、文字列 $U$ が正しい括弧列であるとは、次のいずれかを満たすことをいいます。

- $U$ は空文字列である。
- ある正しい括弧列 $V$ が存在して、$U = \texttt{(} + V + \texttt{)}$ と表せる。
- ある正しい括弧列 $V, W$ が存在して、$U = V + W$ と表せる。

なお、クエリはすべてオンラインで処理してください。

## 制約

- $1 \leq N, Q \leq 2 \times 10^5$
- $S$ は長さ $N$ の文字列である。
- $S$ の各文字は `'a'`〜`'z'`、`'('`、`')'` のいずれかである。
- クエリは合計 $Q$ 個与えられる。
- 種類 `1` のクエリでは、$1 \leq i \leq N$ かつ $c$ は `'a'`〜`'z'`、`'('`、`')'` のいずれかである。
- 種類 `2` のクエリでは、$1 \leq l \leq r \leq N$ である。
- 入力はすべて整数または文字として与えられる。

## 入力

入力は以下の形式で標準入力から与えられます。

```text
N Q
S
query_1
query_2
...
query_Q
```

各クエリは、次のいずれかの形式で与えられます。

```text
1 i c
```

```text
2 l r
```

## 出力

種類 `2` のクエリごとに、判定結果を 1 行に出力してください。

- 正しい括弧列である場合：`Yes`
- 正しい括弧列でない場合：`No`

余計な空白を出力してはいけません。

## 入力例

```text
8 5
a(b)c)d(
2 1 8
2 2 4
1 5 (
2 1 8
2 2 6
```

## 出力例

```text
No
Yes
No
Yes
```

## 説明

最初のクエリでは、$S[1..8]$ から括弧以外の文字を削除すると `())(` になります。  
これは正しい括弧列ではないため、`No` を出力します。

2 番目のクエリでは、$S[2..4]$ から括弧以外の文字を削除すると `()` になります。  
これは正しい括弧列であるため、`Yes` を出力します。
